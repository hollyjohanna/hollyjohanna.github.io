3d modelling

This is a futuristic city I modelled as part of a university assignment. The assignment was to create a street and transform it into a futuristic version of the same street.

This is Cuba Street in Wellington. In the future, it is still a bus lane.

still life

This was a university project to constructed a still life image from possessions that created a story about myself. I was tasked with finding the essence of my personality and model and arrange the objects as realistically and accurately as possible.

These are some of my favourite objects - my best glasses, perfume, and nail polishes.

fusion 360

These are also for a University project. We used Fusion 360 so we could 3D print the end results.

The first project is of a new-age seed, which was meant to be created with organic shapes but still show that it had been designed and engineered. The second project is a drum, which was created as part of an exploration of 3D printing an instrument.
